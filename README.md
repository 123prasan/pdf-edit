# PDF Editor (Web) - Prototype

This is a starter scaffold for a web-based PDF editor using Vite, React, Material UI and PDF.js.

Quick start:

```bash
npm install
npm run dev
```

What's included:
- React + TypeScript
- Material UI theme
- Basic PDF viewer using `pdfjs-dist`

Next steps:
- Implement text editing, annotations, OCR integration, save/export flows
- Add state management and file/version handling
